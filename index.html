allwincnc - it's open source CNC software for the Allwinner H3/H5/H6 based SBCs (single board computers). Project uses Armbian as OS and LinuxCNC as CNC software. Also we are using Allwinner H3/H5/H6's built-in RISC CPU core to handle output/input pulses up to 500 kHz. You can use this project to control any CNC/laser/engraving/3D machine using a display, keyboard and mouse (or remote desktop via network). See project's sources here. To begin, read the installation guide. See a quick video (below) to understand how to use this project for your needs. H5/H6 support is WIP (work in progress).
