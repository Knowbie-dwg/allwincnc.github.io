h3cnc - it's open source CNC software for the Allwinner H3 based SBCs (single board computers). Project uses Armbian as OS and LinuxCNC as CNC software. Also we are using Allwinner H3's built-in RISC CPU core to handle output/input pulses up to 500 kHz. You can use this project to control any CNC/laser/engraving/3D machine using a display, keyboard and mouse (or remote desktop via network). See project's sources here.
